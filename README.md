# 📰 News Central

A modern, responsive **multi-language news aggregator** that brings together popular newspapers from English, Kannada, Hindi, and Telugu languages in one beautiful interface.

## ✨ Features

- Supports 4 languages: **English, Kannada, Hindi & Telugu**
- Clean card-based responsive design
- Sidebar navigation for easy browsing
- One-click access to official newspaper websites
- Built with pure HTML, CSS & Vanilla JavaScript
- Fully mobile-friendly

## 🗞️ Supported Languages & Newspapers

- **English**: The Hindu, Times of India, Indian Express, Hindustan Times, Deccan Herald, Economic Times
- **Kannada**: Prajavani, Vijaya Karnataka, Udayavani, Kannada Prabha, Vijayavani, Vartha Bharati, Samyukta Karnataka
- **Hindi**: Dainik Jagran, Dainik Bhaskar, Hindustan, Amar Ujala, Rajasthan Patrika, Navbharat Times, Punjab Kesari
- **Telugu**: Eenadu, Sakshi, Andhra Jyothi, Namasthe Telangana, Andhra Bhoomi, Prajasakti, Suryaa
